# SwimTracker
API install requirements:  
>1. Python 3.7.2 or above  
```
https://www.python.org/ftp/python/3.7.2/python-3.7.2.exe
```  
>2. Flask  
```
$ pip install flask
```
>3. PyMySQL
```
$ pip install pymysql
```
>4. bcrypt
```
$ pip install bcrypt
```
>5. flask_jwt_extended
```
$ pip install flask_jwt_extended
```
>6. email
```
$ easy_install --upgrade setuptools
$ easy_install email
```
